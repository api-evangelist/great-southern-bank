# Great Southern Bank (great-southern-bank)

Great Southern Bank is one of Australia's largest customer-owned (mutual) banks, operating as a business name of Credit Union Australia Ltd (ABN 44 087 650 959, AFSL and Australian Credit Licence 238317). Formerly Credit Union Australia (CUA) before its 2021 rebrand, it is owned by its customers rather than shareholders and serves retail and small-business members. As an Australian authorised deposit-taking institution (ADI) it participates in the Consumer Data Right (CDR / Open Banking) as a data holder, exposing public, unauthenticated Product Reference Data (PRD) APIs that conform to the Data Standards Body (DSB) Consumer Data Standards (CDS).

## APIs

### Great Southern Bank CDR Product Reference Data API

Public, unauthenticated Consumer Data Right (CDR) Product Reference Data (PRD) API for Great Southern Bank's retail banking products. Confirmed live (HTTP 200, `x-v: 3`, 75 products) and conforms to the DSB Consumer Data Standards banking schema.

- **Human URL:** [https://www.greatsouthernbank.com.au/consumer-data-policy](https://www.greatsouthernbank.com.au/consumer-data-policy)
- **Base URL:** `https://api.open-banking.greatsouthernbank.com.au`
