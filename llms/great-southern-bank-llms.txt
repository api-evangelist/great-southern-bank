# Great Southern Bank

> Great Southern Bank (a business name of Credit Union Australia Ltd, ABN 44 087 650 959) is one of Australia's largest customer-owned banks. Its only public, self-serve API surface is the Consumer Data Right (CDR / Open Banking) Product Reference Data (PRD) API, exposed as a registered CDR data holder for both its retail and Business+ brands. Authenticated consumer data sharing is available only to ACCC-accredited data recipients (ADRs); the bank runs no public developer portal.

## APIs

- [Great Southern Bank CDR Product Reference Data API (retail)](https://api.open-banking.greatsouthernbank.com.au/cds-au/v1/banking/products): Public, unauthenticated product data (HTTP 200, x-v 3, ~75 products).
- [Great Southern Bank Business+ CDR Product Reference Data API](https://od1.open-banking.business.greatsouthernbank.com.au/api/cds-au/v1/banking/products): Public, unauthenticated business product data (HTTP 200, x-v 4/5, ~16 products).
